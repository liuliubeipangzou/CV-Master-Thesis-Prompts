# CV 硕士论文 Prompt 通用模板

这个模板适合在开始某一章写作前快速整理上下文。你可以先填完下面的信息，再把整段内容连同具体任务一起发给大模型。

## 1. 论文基础信息

```text
论文题目：
研究方向：
核心任务：
应用场景：
目标院校/学院：
论文语言：
```

## 2. 研究问题定义

```text
研究背景：
当前方法存在的问题：
我希望解决的核心痛点：
本文方法的主要思路：
与已有方法相比的差异：
```

## 3. 方法与实验信息

```text
方法名称：
Backbone / 基础模型：
核心模块 1：
核心模块 2：
损失函数 / 优化策略：
训练数据集：
测试数据集：
评价指标：
实验提升：
消融实验设计：
可视化结果要点：
```

## 4. 写作约束信息

```text
目标章节：
期望篇幅：
语言风格：
是否保留引用占位符（如 [?]）：
是否需要 LaTeX 公式：
是否需要三级标题：
是否需要先给提纲再展开正文：
特别需要避免的问题：
```

## 5. 推荐总控 Prompt

```text
**Role:** 你是一位熟悉计算机视觉、中文硕士学位论文规范和盲审标准的学术写作顾问。

**Task:** 请基于我提供的信息，为我的论文生成高质量内容。目标是输出逻辑严密、表述规范、能够体现硕士论文工作量与创新性的学术文本。

**Context:**
- 论文题目：[填写]
- 研究方向：[填写]
- 核心任务：[填写]
- 应用场景：[填写]
- 研究背景：[填写]
- 当前方法存在的问题：[填写]
- 我希望解决的核心痛点：[填写]
- 本文方法的主要思路：[填写]
- 与已有方法相比的差异：[填写]
- 方法名称：[填写]
- Backbone / 基础模型：[填写]
- 核心模块 1：[填写]
- 核心模块 2：[填写]
- 损失函数 / 优化策略：[填写]
- 训练数据集：[填写]
- 测试数据集：[填写]
- 评价指标：[填写]
- 实验提升：[填写]
- 消融实验设计：[填写]
- 可视化结果要点：[填写]
- 目标章节：[填写]
- 期望篇幅：[填写]
- 语言风格：[填写]
- 是否保留引用占位符：[填写]
- 是否需要 LaTeX 公式：[填写]
- 是否需要三级标题：[填写]
- 是否需要先给提纲再展开正文：[填写]
- 特别需要避免的问题：[填写]

**Constraints:**
1. 使用中文硕士学位论文的学术表达，避免口语化、营销化和空泛表述。
2. 必须突出问题定义、方法动机、技术细节和实验结论，不写无信息量套话。
3. 如果存在逻辑跳跃，请主动补足因果链条和过渡句。
4. 如果内容涉及实验分析，请同时说明“为什么有效”。
5. 如果信息不足，请优先基于现有上下文做保守推断，并明确指出推断部分。

**Output Format:**
- 第 1 部分：输出目标内容
- 第 2 部分：列出可继续追问优化的 3 个方向
```
